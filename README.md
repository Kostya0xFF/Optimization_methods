# Optimization_methods
Лабораторные работы по предмету "Методы оптимизации".
#### Папка `method_1`
В этой папке находится реализация метода градиентного спуска с правилом Армихо.
#### Папка `method_2`
В этой папке находится реализация градиентного метода с одномерным поиском шага методом дихотомии.
#### Папка `method_3`
В этой папке находится реализация градиентного метода с дроблением шага.
#### Папка `method_4`
В этой папке находятся реализации методов золотого сечения и Фибоначчи для одномерной минимизации.
#### Папка `method_5`
В этой папке находится реализация метода сопряжённых градиентов Флетчера–Ривса для минимизации квадратичной функции.
#### Папка `method_6`
В этой папке находится реализация метода Ньютона–Рафсона для безусловной минимизации функции двух переменных.
